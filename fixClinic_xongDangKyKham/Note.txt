các file .py
- dao.py nơi xử lý liên quan đến database như truy vấn, thêm xóa sửa
- forms.py nơi quy định các ràng buộc forms bằng thư viên wtform
- index -> routes
- utils -> các hàm tiện ích xử lý linh tinh
- test -> test code nếu không chắc chắn

về Templates: Giao diện
> admin:
  - index: giao diện trang chủ của admin được ghi đè

> appointment: Nơi lưu trữ giao diện liên quan tới lịch khám
  - register_appointment: bệnh nhân và y tá đều có thể tham gia
  - search_appointment: chỉ có y tá mới tham gia tác động, tìm được id bệnh nhân mới chuyển trang đăng ký

> auth: xử lý liên quan tới chứng thực người dùng
  - change_password: Nhập password để gửi request lên hệ thống thay đổi password sau khi nhận mail yêu cầu reset pasword
  - login: phần đăng nhập
  - register: đăng ký
  - reset_password: cho người dùng điền mail để phía bên server gửi mail đến client thông qua token được mã hóa đường dẫn
